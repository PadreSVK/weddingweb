# Photos

Drop a photo for each story chapter here:

- `story-1.jpg` — How we met
- `story-2.jpg` — Our travels
- `merlin.jpg` — Merlin (our dog)
- `always.jpg` — "Always" (HP / the tree from the invitation)

You can change the names (or paths) in `config.json` under `story.chapters[*].image`.

Recommended format: JPG, 4:5 ratio (portrait), at least 800 px wide.
If a photo is missing, an emoji placeholder is shown in its place.
